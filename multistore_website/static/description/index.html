Multistore Website

The Multistore website Plugin can be used to manage features on unlimited Odoo stores from one single Odoo admin.


Manage and publish store across an Odoo multi-site environment with multiple web shops in different domains or sub-domains. There is the main admin setting where you can manage catalog, eCommerce Categories, Blog, users, menus, payments methods & shipping methods.


Instead of working with multiple separate databases based Odoo and spending a fortune on the license purchase, take advantage of our cutting-edge multi-store shopping cart and enjoy unmatched convenience and flexibility of managing all your websites from a single reliable location on a single database.

List of Features


  • Multiple websites/store in a single database
  • Easy to add new websites/store
  • Easy to create website wise menus
  • Use any domains for websites (not only subdomains)
  • Multi-Websites with same or different Company
  • Different Payment Methods for different Websites
  • Different Shipping Methods for different Websites
  • Different eCommerce Categories for different Websites
  • Different Products for different Websites
  • Different Blog and post for different Websites
  • Added website filter in Sales Order
  • Added website filter in User(Same email not allow to multile website)
  • Different Logo and Favicon for different Websites
  • Easy to add Cookies and reCaptcha website base
  • Different Social Media links for different Websites
  • Different Salesperson and Sales Channel for different Websites
  • Default language selection for website base
  • Different theme support for each website

Notes: Multiple theme feature for odoo themes (only) like theme bootswatch.Not allow to install two theme which are depend on any other modules like website_sale because it will conflict with other themes
